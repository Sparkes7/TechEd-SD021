Box Sizing with Dogs!

The dog pictures have a width of 25%, and there are four dogs. These SHOULD fit in the web page, taking 100% of the width, but they do not because the 10px red border takes up space that is not accounted for in the 25% width

The below dogs have the "Box Sizing" property set to "border-box" which takes into account the border within the 25% width, and therefore the four 25% width dogs fit perfectly into the width, totalling 100% of the page width
